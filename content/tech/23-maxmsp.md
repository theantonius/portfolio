Max/MSP is a visual programming environment for interactive audio and media, and I use it to connect physical inputs, sensors, MIDI controllers, and cameras to sound and visuals in real time. It's been part of my toolkit since studying at ITP, where real-time media processing was central to the program. I reach for it on live performance systems, installation work, and any project where audio and physical computing need to talk to each other.
